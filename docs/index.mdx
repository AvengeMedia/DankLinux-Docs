Dank Linux is a desktop environment and suite of tools and desktop applications designed to provide a clean, efficient, and highly customizable modern desktop experience on Wayland compositors.

## Quick Start

*This will set up Dank Linux on your system, including all dependencies, the DankMaterialShell desktop shell, and default configurations.*

```bash
curl -fsSL https://install.danklinux.com | sh
```

Supports Arch Linux + Derivatives, Fedora + Derivatives, Ubuntu, Debian, and openSUSE.

## The Suite

- **[DankMaterialShell](dankmaterialshell/)** - A feature-rich desktop shell built with [Quickshell](https://quickshell.org/) and [Go](https://golang.org/).
- **[DankSearch](danksearch/)** - Native file system search with fuzzy search capabilities, built into DMS
- **[DGOP](dgop/)** - A comprehensive system monitoring and operations platform with CLI and REST API
- **dcal (in development)** - A calendar application that integrates with online calendar services.
- **[dankinstall](danklinux/)** - An automated installer and management tool for the entire Dank Linux suite

## Beautiful by Default

DankMaterialShell provides a stunning desktop experience right out of the box:
- **20+ Customizable Widgets**: App launcher, workspace switcher, media player, clock, weather, system tray, notifications, and more
- **Dynamic Theming**: Automatic color schemes generated from your wallpaper using matugen
- **Smooth Animations**: Polished transitions and effects throughout the desktop
- **Material Design**: Modern, clean aesthetics inspired by Material Design 3

## Powerful System Monitoring

DGOP provides comprehensive system insights and operations capabilities:
- **Real-time Metrics**: CPU, memory, network, disk, and GPU monitoring with detailed statistics
- **Advanced CLI**: Powerful command-line tools with filtering, sorting, and search capabilities
- **REST API**: Complete JSON API with OpenAPI documentation for programmatic access
- **Process Management**: Advanced process monitoring with real-time updates and control
- **Hardware Monitoring**: Temperature sensors, hardware information, and performance tracking
- **Network Analysis**: Interface statistics, connection monitoring, and traffic analysis

## Easy Installation

The installer handles everything automatically:
- Detects your Linux distribution
- Installs all necessary dependencies
- Configures your compositor (niri or Hyprland)
- Sets up theming and default configurations

## Supported Platforms

Dank Linux works on:
- **Arch Linux** and derivatives (Manjaro, EndeavourOS, CachyOS)
- **Fedora** 40+ and derivatives (Nobara, Fedora Asahi Remix)
- **Ubuntu** 25.04+
- **Debian** 13+ (Trixie)
- **openSUSE Tumbleweed**
- **NixOS** (via flake)

## Optimized for Wayland

Built specifically for modern Wayland compositors:
- **niri** - Scrollable-tiling compositor (primary support)
- **Hyprland** - Dynamic tiling compositor (fully supported)
- **Generic Wayland** - Basic support for other compositors
