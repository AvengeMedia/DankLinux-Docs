The **DankLinux Repository** provides pre-built packages for DankMaterialShell and related tools across multiple distributions through Open Build Service (OBS) and Launchpad PPA.

## Overview

DankMaterialShell is now available on more distributions than ever before! We've expanded beyond Arch to include:

- **Fedora** (via COPR)
- **Debian** (via Open Build Service)
- **Ubuntu** (via Launchpad PPA)
- **OpenSUSE** (via Open Build Service)

These repositories are maintained by the AvengeMedia team and provide both stable releases and bleeding-edge development builds.

## What's Included

The DankLinux repositories provide:

- **DankMaterialShell (dms)**: The complete desktop shell with widgets and CLI tools
- **dgop**: System telemetry daemon for resource monitoring
- **dsearch**: Filesystem search engine
- [**niri**](https://github.com/YaLTeR/niri): Scrollable - Tiling Wayland compositor
- [**Quickshell**](https://github.com/quickshell-mirror/quickshell): The underlying framework (when not available in official repos)
- **Related Dependencies**: Optional components like `cliphist`, `matugen`, and more

## Package Variants

Each repository provides two variants:

### Stable Release (`dms`)

- Tagged releases with tested features
- Recommended for production use
- Updated with major version releases

### Development Build (`dms-git`)

- Latest commits from the master branch
- Cutting-edge features and fixes
- May contain experimental changes

## GitHub Repository

The packaging configurations and build scripts are maintained in the official DankLinux repository:

**Repository**: [github.com/AvengeMedia/DankLinux](https://github.com/AvengeMedia/DankLinux)

This repository contains:
- OBS packaging specifications (`.spec` files, build configs)
- Debian packaging files (`debian/` directories)
- CI/CD workflows for automated builds
- Release management tooling
