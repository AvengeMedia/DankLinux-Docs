# Getting Started

Welcome to Dank Linux! This guide will help you get started with installing and configuring your modern Wayland desktop environment.

## What is Dank Linux?

Dank Linux is a comprehensive desktop environment suite that includes:

- **DankMaterialShell (dms)** - A modern Wayland desktop shell with 20+ customizable widgets
- **dgop** - A powerful system monitoring tool with CLI and REST API
- **dankinstall** - An automated installer that sets everything up for you

## Quick Installation

The fastest way to get started is with our one-line installer:

```bash
curl -fsSL https://install.danklinux.com | sh
```

This will:
- Install all dependencies for your distribution
- Set up DankMaterialShell
- Configure your chosen compositor (niri or Hyprland)
- Apply default themes and configurations

## Supported Distributions

Dank Linux officially supports:

- **Arch Linux** and derivatives (Manjaro, EndeavourOS, CachyOS, etc.)
- **Fedora** and derivatives (Nobara, Fedora Asahi Remix)
- **Ubuntu** 25.04+
- **Debian** 13+ (Trixie, niri-only)
- **openSUSE Tumbleweed**
- **NixOS** (via separate flake)

## System Requirements

- **Operating System**: A supported Linux distribution
- **Display Server**: Wayland
- **Compositor**: niri or Hyprland (recommended)
- **Go**: Version 1.24+ (for building from source)
