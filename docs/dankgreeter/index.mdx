# DankGreeter

`dms-greeter` is a [greetd](https://github.com/kennylevinsen/greetd) greeter using the same aesthetics as the DankMaterialShell lock screen. It is a single binary with the UI embedded, so there is nothing else to install besides greetd, quickshell, and a compositor.
