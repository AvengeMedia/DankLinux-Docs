# DGOP Configuration

## API

For the API, you can set the following environment variable to override the port of the http server:
- `API_PORT` (defaults to `63484`)

## Troubleshooting

### Common Issues

**API not starting:**
- Check port availability: `netstat -tlnp | grep :63484`
- Check logs
