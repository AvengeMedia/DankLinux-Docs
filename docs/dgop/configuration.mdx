# DGOP Configuration

Configure DGOP settings for monitoring behavior, API options, and system integration.

## Configuration File

DGOP settings are stored in `~/.config/dgop/config.json`:

```json
{
  "api": {
    "enabled": true,
    "port": 8080,
    "host": "localhost",
    "auth": false,
    "username": "",
    "password": ""
  },
  "monitoring": {
    "refresh_interval": "1s",
    "max_history": 1000,
    "enable_gpu": true,
    "enable_temperature": true
  },
  "logging": {
    "level": "info",
    "file": "/var/log/dgop.log"
  }
}
```

## API Configuration

### Basic API Settings

Configure the REST API server:

```json
{
  "api": {
    "enabled": true,
    "port": 8080,
    "host": "0.0.0.0"
  }
}
```

### Authentication

Enable API authentication:

```json
{
  "api": {
    "auth": true,
    "username": "admin",
    "password": "secure_password"
  }
}
```

### CORS Settings

Configure Cross-Origin Resource Sharing:

```json
{
  "api": {
    "cors": {
      "enabled": true,
      "origins": ["http://localhost:3000", "https://yourdomain.com"]
    }
  }
}
```

## Monitoring Configuration

### Refresh Intervals

Set how often metrics are updated:

```json
{
  "monitoring": {
    "refresh_interval": "1s",
    "api_refresh_interval": "5s",
    "widget_refresh_interval": "2s"
  }
}
```

### Data Retention

Configure how much historical data to keep:

```json
{
  "monitoring": {
    "max_history": 1000,
    "history_cleanup_interval": "1h"
  }
}
```

### Feature Toggles

Enable/disable specific monitoring features:

```json
{
  "monitoring": {
    "enable_cpu": true,
    "enable_memory": true,
    "enable_disk": true,
    "enable_network": true,
    "enable_gpu": true,
    "enable_temperature": true,
    "enable_processes": true
  }
}
```

## Logging Configuration

### Log Levels

Set the logging verbosity:

```json
{
  "logging": {
    "level": "info",
    "file": "/var/log/dgop.log",
    "max_size": "100MB",
    "max_backups": 3
  }
}
```

Available log levels: `debug`, `info`, `warn`, `error`

### Log Rotation

Configure log file rotation:

```json
{
  "logging": {
    "rotation": {
      "enabled": true,
      "max_size": "100MB",
      "max_backups": 3,
      "max_age": "7d"
    }
  }
}
```

## CLI Configuration

Use the DGOP CLI to manage settings:

```bash
# View current configuration
dgop config show

# Edit configuration
dgop config edit

# Set specific values
dgop config set api.port 9090
dgop config set monitoring.refresh_interval "2s"

# Reset to defaults
dgop config reset
```

## DMS Integration

### Widget Configuration

Configure desktop widgets:

```json
{
  "dms": {
    "widgets": {
      "cpu_widget": {
        "enabled": true,
        "refresh_interval": "1s",
        "show_cores": true
      },
      "memory_widget": {
        "enabled": true,
        "refresh_interval": "2s",
        "show_swap": true
      },
      "network_widget": {
        "enabled": true,
        "refresh_interval": "5s",
        "interfaces": ["eth0", "wlan0"]
      }
    }
  }
}
```

### IPC Configuration

Configure Inter-Process Communication:

```json
{
  "dms": {
    "ipc": {
      "enabled": true,
      "socket_path": "/tmp/dgop.sock",
      "timeout": "5s"
    }
  }
}
```

## Performance Tuning

### Resource Limits

Limit resource usage:

```json
{
  "performance": {
    "max_cpu_percent": 10,
    "max_memory_mb": 100,
    "worker_threads": 4
  }
}
```

### Caching

Configure data caching:

```json
{
  "performance": {
    "cache": {
      "enabled": true,
      "ttl": "30s",
      "max_size": "50MB"
    }
  }
}
```

## Security Configuration

### API Security

Secure the API endpoint:

```json
{
  "security": {
    "api": {
      "rate_limit": {
        "enabled": true,
        "requests_per_minute": 60
      },
      "ssl": {
        "enabled": false,
        "cert_file": "",
        "key_file": ""
      }
    }
  }
}
```

### Access Control

Configure access restrictions:

```json
{
  "security": {
    "access": {
      "allowed_ips": ["127.0.0.1", "192.168.1.0/24"],
      "blocked_ips": []
    }
  }
}
```

## Environment Variables

Override configuration with environment variables:

```bash
# API settings
export DGOP_API_PORT=9090
export DGOP_API_HOST=0.0.0.0

# Monitoring settings
export DGOP_REFRESH_INTERVAL=2s
export DGOP_MAX_HISTORY=500

# Logging settings
export DGOP_LOG_LEVEL=debug
export DGOP_LOG_FILE=/var/log/dgop.log
```

## Troubleshooting

### Common Issues

**API not starting:**
- Check port availability: `netstat -tlnp | grep :8080`
- Verify configuration: `dgop config validate`
- Check logs: `dgop logs`

**High CPU usage:**
- Increase refresh interval
- Disable unnecessary features
- Check worker thread count

**Memory issues:**
- Reduce max_history
- Enable data cleanup
- Check cache settings

### Debug Mode

Enable debug logging:

```bash
# Start with debug logging
dgop serve --log-level debug

# Validate configuration
dgop config validate

# Test configuration
dgop test-config
```
