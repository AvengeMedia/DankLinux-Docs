# Keyboard & Navigation

The calendar window is fully keyboard-driven. Press <kbd>?</kbd> at any time to bring up the in-app shortcut overlay.

## Views

Five views are available. Switch between them from the sidebar, or with the keys below.

| View       | Description                                              |
| ---------- | -------------------------------------------------------- |
| **Month**  | A full month grid (the default view)                     |
| **Week**   | A seven-day time grid                                    |
| **Day**    | A single-day time grid                                   |
| **Agenda** | A scrolling list of upcoming days                        |
| **Tasks**  | Your VTODO task lists ‚ÄĒ shown when *Show tasks* is enabled and an account provides them |

Your last-used view is remembered across sessions. You can also open a specific view from the CLI, e.g. `dcal show --view week`.

## Moving around

- <kbd>t</kbd> jumps back to **today**.
- <kbd>h</kbd>/<kbd>l</kbd> (or <kbd>‚Üź</kbd>/<kbd>‚Üí</kbd>) move one **day** at a time.
- <kbd>[</kbd>/<kbd>]</kbd> (or <kbd>PgUp</kbd>/<kbd>PgDn</kbd>) move by one **period** ‚ÄĒ a day, week, or month depending on the current view.
- <kbd>&#123;</kbd>/<kbd>&#125;</kbd> jump a full **year** back or forward.
- <kbd>g</kbd> opens the **Go to date** dialog for larger jumps.

### Go to date

Press <kbd>g</kbd> to open a compact go-to dialog for jumping across long distances in one step. You can either:

- **Type a date** and press <kbd>Enter</kbd>. Several formats are understood, including `2026-07-13` (ISO), `2026/07/13`, `2026.07.13`, `2026-07` (first of the month), and `2026` (start of the year). Many written forms such as `Jul 13 2026` also work. As you type a recognized date, the picker previews it.
- **Use the mini picker.** The single chevrons step by month and the double chevrons step by year; click a day to jump straight to it.

## Selecting and opening events

In day, week, and agenda views, <kbd>j</kbd>/<kbd>k</kbd> (or <kbd>‚Üď</kbd>/<kbd>‚ÜĎ</kbd>) walk through events chronologically, crossing into the next or previous day that has events. In month view the same keys move by week.

- <kbd>Tab</kbd>/<kbd>Shift+Tab</kbd> jump to the next or previous **day that has events**.
- <kbd>Enter</kbd> opens the selected event, or creates a new one if nothing is selected.
- <kbd>Esc</kbd> clears the current event selection.
- <kbd>c</kbd> creates an event on the selected day.

## Search

Press <kbd>/</kbd> or <kbd>Ctrl+F</kbd> to open spotlight-style search. Type at least two characters to search across every connected calendar; selecting a result jumps to and opens that event.

## Shortcut reference

### Navigation

| Keys | Action |
| ---- | ------ |
| <kbd>t</kbd> | Go to today |
| <kbd>h</kbd> / <kbd>‚Üź</kbd> | Previous day |
| <kbd>l</kbd> / <kbd>‚Üí</kbd> | Next day |
| <kbd>j</kbd> / <kbd>‚Üď</kbd> | Next event (next week in month view) |
| <kbd>k</kbd> / <kbd>‚ÜĎ</kbd> | Previous event (previous week in month view) |
| <kbd>[</kbd> / <kbd>PgUp</kbd> | Previous day, week, or month |
| <kbd>]</kbd> / <kbd>PgDn</kbd> | Next day, week, or month |
| <kbd>&#123;</kbd> | Previous year |
| <kbd>&#125;</kbd> | Next year |
| <kbd>g</kbd> | Go to date |
| <kbd>Tab</kbd> | Next day with events |
| <kbd>Shift+Tab</kbd> | Previous day with events |
| <kbd>Enter</kbd> | Open or create event |
| <kbd>Esc</kbd> | Clear event selection |

### Views

| Keys | Action |
| ---- | ------ |
| <kbd>d</kbd> / <kbd>Ctrl+1</kbd> | Day view |
| <kbd>w</kbd> / <kbd>Ctrl+2</kbd> | Week view |
| <kbd>m</kbd> / <kbd>Ctrl+3</kbd> | Month view |
| <kbd>a</kbd> / <kbd>Ctrl+4</kbd> | Agenda view |
| <kbd>Ctrl+5</kbd> | Tasks view |

### Sidebar

Press <kbd>s</kbd> to focus the sidebar; press <kbd>s</kbd> or <kbd>Esc</kbd> to leave it.

| Keys | Action |
| ---- | ------ |
| <kbd>j</kbd> / <kbd>k</kbd> | Move through sidebar items |
| <kbd>h</kbd> / <kbd>l</kbd> | Collapse or expand a section |
| <kbd>Enter</kbd> / <kbd>Space</kbd> | Toggle calendar, switch view, or sync account |
| <kbd>r</kbd> | Rename calendar |
| <kbd>Del</kbd> | Remove calendar or account |

### General

| Keys | Action |
| ---- | ------ |
| <kbd>c</kbd> | Create event |
| <kbd>/</kbd> / <kbd>Ctrl+F</kbd> | Search |
| <kbd>Ctrl+,</kbd> | Settings |
| <kbd>?</kbd> | Toggle the shortcut overlay |

:::note
Under right-to-left locales the calendar mirrors automatically, and the horizontal navigation keys (<kbd>h</kbd>/<kbd>l</kbd>, <kbd>‚Üź</kbd>/<kbd>‚Üí</kbd>) follow the mirrored direction.
:::
