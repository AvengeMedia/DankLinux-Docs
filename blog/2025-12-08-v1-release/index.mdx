We're excited to announce DankMaterialShell (DMS) v1.0! What started as a small hobby project has grown into a full desktop shell and application suite with thousands of users and dozens of contributors.

This release marks more than just a version number, it's a commitment to stability and predictability. Going forward, DMS will follow a milestone/roadmap-based release cycle, with bug-fix releases as needed, as we continue to mature the project and expand the community and ecosystem.

## What's New

### OMEGA Bar

Add up to **4** bar configurations, each with their own set of widgets, layout, and style. Configure which monitor each bar appears on and independently control each bar with new IPCs.

### Packages for Debian, Ubuntu, OpenSUSE, Fedora, and Arch

DMS is now available on Debian, Ubuntu, OpenSUSE, Fedora, and Arch Linux - via OBS, PPA, COPR, and Arch User Repository (AUR).

### Available on nixpkgs

DMS is now available on nixpkgs. This simplifies installation significantly for NixOS users and those using Nix package manager and represents a major step towards broader adoption.

Special thanks to [@LuckShiba](https://github.com/LuckShiba) for creating the nixpkgs package and [marcusramberg](https://github.com/marcusramberg) for approving!

### Dank Color Picker

The color picker is not new, but the eye dropper/color picker tool is now built-in to DMS - no more need for third-party tools! It is now integrated into DMS and available as a standalone tool for Wayland compositors. See the [CLI color picker documentation](https://danklinux.com/docs/dankmaterialshell/cli-color-picker).
