## What's New

### Desktop Widgets & Plugins

<!-- TODO: Screenshot -->

DMS 1.2 introduces an extremely powerful Desktop Widget system, there are some built-in widgets (dgop, clock) and already several community-developed desktop widgets available in the [DMS Widget Registry](https://danklinux.com/plugins).

### Theme Registry

<!-- TODO: Screenshot -->

DMS 1.2 introduces an extremely powerful theme system where a single theme can have numerous variants and acccent colors. The plugin registry now includes a [Theme Registry](https://danklinux.com/plugins?tab=themes) with numerous community-developed themes available for easy installation. Learn how to [contribute your own themes](https://github.com/AvengeMedia/dms-plugin-registry/blob/master/CONTRIBUTING.md#contributing-a-theme).

### Native Clipboard and History

<!-- TODO: Screenshot -->

Full, zero-dependency clipboard and clipboard history integration.
- No dependency on tools such as `wl-clipboard` or `cliphist`, they can be uninstalled.
- Configurable size, retention durations, and automatic cleanup.
- Automatically ignores data from  password managers (such as KeepassXC and 1Password) for security.

### Monitor Configurator

<!-- TODO: Screenshot -->

Configure display settings directly from DMS Settings. Available for niri, Hyprland, and MangoWC. Includes positioning, resolution, refresh rate, VRR, HDR (Hyprland), scaling, mirroring, and more.

### Notification Improvements

- Swipe to dismiss notification popups and from notification center
- Notification persistence & history
- Configurable notification display on lock screen

### Settings Search

<!-- TODO: Screenshot -->

Quickly find settings with the new search functionality.

### DMS Doctor

New `dms doctor` command for diagnosing configuration issues and system compatibility.

### Compositor Overrides

Configure gaps, window rules, and border settings for niri, Hyprland, and MangoWC directly from DMS.

### Initial RTL & i18n Support

Groundwork for right-to-left language support and internationalization.

This release contains changes that may require manual intervention.
